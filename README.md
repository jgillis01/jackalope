# Rack::Amqp

Server to run your rack (rails) application using AMQP as the transport
protocol.

## Installation

Add this line to your application's Gemfile:

    gem 'rack-amqp'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install rack-amqp

## Usage

rabbicorn <rackup file>

for rails app:

rabbicorn config.ru
